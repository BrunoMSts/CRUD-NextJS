### To run this project you will need!
- Clone this repo to your local machine
  ```
  git clone https://github.com/BrunoMSts/CRUD-NextJS.git
  ```
- Run npm or yarn
  ```javascript
  npm install || yarn install
  ```
- Run backend
  ```javascript
  npm start || yarn start
  ```
- Run frontend
  ```javascript
  npm dev || yarn dev
  ```
